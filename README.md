# 💰 Finance App - Controle Financeiro com Django + Docker

Aplicação web desenvolvida com **Django** para controle financeiro pessoal, permitindo o gerenciamento de receitas, despesas, categorias e formas de pagamento.

O projeto utiliza **Docker** para padronizar o ambiente de desenvolvimento e execução.

---

# 🚀 Funcionalidades

## 🔐 Autenticação
- Registro de usuários
- Login e logout
- Proteção de rotas com autenticação

## 📊 Lançamentos Financeiros
- Criar, editar, listar e excluir lançamentos (CRUD completo)
- Classificação por:
  - Tipo (Receita / Despesa)
  - Categoria
  - Forma de pagamento
- Controle de datas
- Suporte a lançamentos parcelados

## 🗂 Categorias
- Criação de categorias personalizadas por usuário

## 💳 Formas de Pagamento
- Cadastro de formas de pagamento (Pix, cartão, etc.)

## 🎨 Interface
- Templates iniciais com CSS/Bootstrap/Tailwind

---

# 🛠 Tecnologias Utilizadas

- Python 3.11
- Django 4.x
- PostgreSQL
- Docker
- Docker Compose
- Tailwind
